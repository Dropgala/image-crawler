## Details about the project:

- React library.
- Nextjs framework.
- Styled-components.
- PWA using next-offline.
- SEO using next-seo.

## Getting Started

First, install all dependencies for the project:

```bash
yarn install
```

Second, run the development server:

```bash
yarn dev
```
